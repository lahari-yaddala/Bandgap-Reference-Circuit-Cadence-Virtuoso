# Bandgap-Reference-Circuit-Cadence-Virtuoso
# Bandgap Reference (BGR) Circuit Design using Cadence Virtuoso

This project presents the design and simulation of a **Bandgap Reference (BGR) Circuit** using **Cadence Virtuoso**. The objective is to generate a stable reference voltage that remains nearly constant across variations in temperature, power supply, and process conditions.

The project was completed during my **Analog IC Design Internship** and demonstrates the fundamentals of analog circuit design and simulation.

---

## 📌 Project Overview

A Bandgap Reference (BGR) circuit is one of the most fundamental building blocks in Analog and Mixed-Signal IC design. It provides a precise reference voltage for analog and digital integrated circuits by combining complementary temperature characteristics.

This project includes circuit design, simulation, and performance analysis using Cadence Virtuoso.
